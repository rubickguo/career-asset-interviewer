# 仙人指路：职业资产深访

[English README](README.md)

仙人指路是一个 Codex skill，用来做深度职业资产访谈。它不是普通的“简历优化器”，而是先帮用户澄清职业方向，再从过往经历里提炼证据，最后生成简历、JD 策略、面试故事、个人网站和作品集素材。

这个 skill 的核心判断是：**不要从简历开始，而要先从人开始**。先弄清楚用户想做什么、不想做什么、擅长什么、喜欢什么，以及哪些“不想做”其实只是预设了困难；然后再把简历作为证据库来深挖。

## 它能做什么

- 通过结构化对话澄清职业意图
- 区分真实不喜欢、坏经验、硬限制和可解决的预设困难
- 结合喜欢、擅长、过往经历、市场机会和转换成本，排序推荐职业方向
- 在改简历前先确定核心职业关键词
- 围绕关键词深挖简历里的项目和经历
- 沉淀可复用的职业资产库
- 输出简历、JD 策略、面试故事、自我介绍、个人网站文案和作品集素材

## 核心流程

1. 职业意图访谈
2. 偏好、能力、限制澄清
3. 职业方向排序
4. 核心关键词确认
5. 简历与项目深挖
6. 职业资产库沉淀
7. 交付物生成

## 使用方式

把这个文件夹安装或复制到 Codex skills 目录后，可以这样显式调用：

```text
Use $career-asset-interviewer，帮我根据这份简历先澄清职业方向，再确定核心关键词。
```

一次典型使用会包括：

- 上传或粘贴旧简历
- 说明当前目标，例如找工作、转行、晋升、重新定位
- 可选：提供目标 JD
- 在改简历前先完成一轮简短的职业意图访谈

## 示例 Prompt

```text
Use $career-asset-interviewer。我有一份旧简历和一个目标 JD，请先帮我判断这个方向是否适合我，再确定简历应该凸显的关键词。
```

## 设计原则

- 先理解人，再处理文档
- 方向和关键词没有定清楚之前，不急着改简历
- 不编造经历、头衔、学历、指标或成果
- 对未确认信息明确标注“待确认”
- 每次只围绕一个主题追问，避免用户疲劳
- 简历、网站、面试回答都是职业资产库的下游产物

## 可生成的交付物

根据用户目标，这个 skill 可以输出：

- 职业定位总结
- 职业方向排序
- 核心关键词地图
- 结构化项目卡片
- 能力证据库
- 简历 bullet 和完整简历草稿
- JD 匹配分析与投递策略
- 面试 STAR 故事库
- 自我介绍
- 个人网站文案或简单 HTML 页面
- 30/60/90 天行动计划
