# Career Asset Schema

Use this shared schema across all career skills. Each skill should update only the sections it owns and preserve confirmed user facts.

## career-assets/profile.md

```markdown

## Current Goal

## Target Directions

## Wants

## Does Not Want

| Item | Reason | Type | Solvable? | Notes |
|---|---|---|---|---|

## Strengths

## Preferences

## Constraints

## Confirmed Facts

## Open Questions
```
