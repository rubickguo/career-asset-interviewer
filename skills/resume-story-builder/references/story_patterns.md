# Story Patterns

## Resume Bullet Formula

```text
在 [业务/产品/团队背景] 下，负责 [范围]，通过 [关键动作/方法]，实现/推动 [结果]，体现 [关键词能力]。
```

```text
主导/参与 [项目]，解决 [问题]；完成 [动作]，带来 [业务/效率/质量/用户/团队影响]。
```

## STAR Structure

- Situation: business context and problem.
- Task: user's responsibility.
- Action: decisions, actions, coordination, tradeoffs.
- Result: measured or qualified impact.

## Ownership Wording

- Use `主导` only when the user had clear ownership.
- Use `负责方案设计` when the user owned the solution but not final business decision.
- Use `核心参与` when the user was important but not the owner.
- Use `推动` when the value was coordination or execution leverage.
